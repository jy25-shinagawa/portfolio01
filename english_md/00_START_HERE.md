## 💡 Useful Tips

### Tip 1: Quick language switching
```javascript
// Open DevTools (F12) and execute:
changeLanguage('ja');  // Japanese
changeLanguage('fr');  // French
changeLanguage('ru');  // Russian
```

### Tip 2: View all errors
```javascript
// In DevTools console:
console.log(document.getElementById('email'));  // Check element
console.log(localStorage.getItem('preferred-lang'));  // Languages
```

### Tip 3: Add your own language
```javascript
// In login-script.js find loginTranslations and add:
newLang: {
    "page-title": "Ваш перевод",
    "login-title": "Ваш перевод",
    // ... остальные ключи
}

// In HTML add:
<option value="newLang">Language</option>
```

### Tip 4: Change colors
```css
/* In login-style.css find :root and change: */
:root {
    --primary: #FF0000;      /* Красный вместо #747106 */
    --secondary: #00FF00;    /* Зеленый вместо #D2BD96 */
}
```

## 📞 If something doesn't work

### Problem 1: Page doesn't load
```
✓ Check that files are in the same folder
✓ Check that extensions are correct (.html, .css, .js)
✓ Use a local server (python -m http.server 8000)
```

### Problem 2: Styles are not applied
```
✓ Clear browser cache (Ctrl+Shift+Delete)
✓ Reload the page (Ctrl+F5)
✓ Check file paths in HTML
✓ Open DevTools (F12) and check for errors
```

### Problem 3: Languages don't switch
```
✓ Open DevTools → Console
✓ Execute: changeLanguage('ja')
✓ If it works - the problem is in HTML attributes
✓ Проверьте все data-i18n атрибуты на наличие
```

### Problem 4: Form doesn't submit
```
✓ Check the console for errors
✓ Make sure all input elements have IDs
✓ Check that JavaScript is loaded
✓ Enter correct data (email and password 6+ characters)
```
