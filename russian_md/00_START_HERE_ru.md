### Совет 1: Быстрое переключение языков
```javascript
// Откройте DevTools (F12) и выполните:
changeLanguage('ja');  // Японский
changeLanguage('fr');  // Французский
changeLanguage('ru');  // Русский
```

### Совет 2: Просмотр всех ошибок
```javascript
// В консоли DevTools:
console.log(document.getElementById('email'));  // Проверить элемент
console.log(localStorage.getItem('preferred-lang'));  // Языки
```

### Совет 3: Добавить свой язык
```javascript
// В login-script.js найти loginTranslations и добавить:
newLang: {
    "page-title": "Ваш перевод",
    "login-title": "Ваш перевод",
    // ... остальные ключи
}

// В HTML добавить:
<option value="newLang">Язык</option>
```

### Совет 4: Изменить цвета
```css
/* В login-style.css найти :root и изменить: */
:root {
    --primary: #FF0000;      /* Красный вместо #747106 */
    --secondary: #00FF00;    /* Зеленый вместо #D2BD96 */
}
```

## 📞 Если что-то не работает

### Проблема 1: Страница не загружается
```
✓ Проверьте, что файлы в одной папке
✓ Проверьте, что расширения правильные (.html, .css, .js)
✓ Используйте локальный сервер (python -m http.server 8000)
```

### Проблема 2: Стили не применяются
```
✓ Очистите кэш браузера (Ctrl+Shift+Delete)
✓ Перезагрузите страницу (Ctrl+F5)
✓ Проверьте пути к файлам в HTML
✓ Откройте DevTools (F12) и проверьте ошибки
```

### Проблема 3: Языки не переключаются
```
✓ Откройте DevTools → Console
✓ Выполните: changeLanguage('ja')
✓ Если работает - значит проблема в HTML атрибутах
✓ Проверьте все data-i18n атрибуты на наличие
```

### Проблема 4: Форма не отправляется
```
✓ Проверьте консоль на ошибки
✓ Убедитесь что все input элементы имеют id
✓ Проверьте, что JavaScript загружен
✓ Введите правильные данные (email и пароль 6+ символов)
```
